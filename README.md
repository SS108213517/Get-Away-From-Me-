# Get-Away-From-Me
# 動機發想  
大多數在外租房子的人對居家安全都會抱有一定程度的疑慮，為此我們希望開發一個能記錄誰在你家門口做了什麼事情的專案。

# 所需設備  
● Raspberry pi 3 （課程提供）   
● 紅外線感測器 （課程提供）  
● Logitch C270 HD WEBCAM （組員提供）
